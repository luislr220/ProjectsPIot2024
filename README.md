# Estación meteorologica
## Integrantes

- Luis Armando Largo Ramirez
- Omar Zuñiga Abundis

## Objetivo general
Este proyecto tiene la finalidad de poder medir la humedad, presion atmosferica y humedad, para funcionar como una estación portatil
de clima.
